## Primeros pasos

1. Activar entorno:
   \`\`\`
   source venv/bin/activate
   \`\`\`
2. Instalar dependencias:
   \`\`\`
   pip install -r requirements.txt
   \`\`\`
3. Configurar \`.env\` basado en \`.env.example\`.
4. Ejecutar:
   \`\`\`
   ./main.py
   \`\`\`
